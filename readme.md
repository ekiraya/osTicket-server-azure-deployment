<h2>List of Prerequisites</h2>
<p>For this tutorial, we need 3 prerequisites: the first one is to have a functioning domain controller system. For a guide on how to create one, refer [to this guide.](https://github.com/ekiraya/Deploying-azure-active-directory/tree/main)</p>
<p>The second one is to have an Azure VM with osTicket installed. We prefer that virtual machine to run Windows Server, but this tutorial can be done with other operating systems as well. If you need a guide on how to install osTicket in a virtual machine, please refer [to this guide.](https://github.com/ekiraya/In_depth_osTicket_Installation)</p>
<p>The third one is to have that vm where osTicket is installed joined into our domain for a guide on how to achieve that, [check this section on the tutorial referenced in step 1 ](https://github.com/ekiraya/Deploying-azure-active-directory/blob/main/README.md#join-computers-to-the-domain)</p>
